# Steering Controller - PC Receiver (Executable)

This repository hosts the compiled executable (`.exe`) and auto-update configuration for the VJoy Steering Controller PC application.

## 🚀 Download & Installation

1. Download the latest `VJoyServer.exe` directly from this repository.
2. Ensure you have [vJoy](https://github.com/shauleiz/vJoy) installed on your PC.
3. Run `VJoyServer.exe`.

## 🔄 Auto-Update System

This application includes a built-in auto-update system. 
- `version.json` acts as the source of truth for the latest version.
- When you open the app, it will check `version.json` on this repository.
- If a new version is detected, you will be prompted to download it seamlessly.
